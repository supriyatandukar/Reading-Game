# 👻 The Forgotten Search

A small interactive horror/mystery game made with **HTML, CSS, and JavaScript**.

## What's it about?

You wake up in a strange foggy forest with absolutely no memory of who you are.
No name.
No family.
No past.
The only thing you know is...

> **Keep searching.**

As you explore different places, you'll uncover forgotten memories and slowly piece together the truth. But be careful! Things aren't always what they seem.

## 🛠️ Built With
* HTML
* CSS
* JavaScript

No frameworks.
Just vibes lol


## ⚠️ Heads Up
This game is story-driven, so take your time reading. There aren't any jump scares (yet), but there are a few creepy moments and a mystery waiting to be solved.

Thanks for checking out my little project!
